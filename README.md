🌿 Plant Hub – AI-Powered E-Commerce Platform 🌿
🛍️ Overview
Plant Hub is a cutting-edge e-commerce platform designed for buying and selling plants. This platform integrates an AI-powered chatbot to assist users with plant care recommendations and real-time query handling. An intuitive Admin Dashboard allows management of products, orders, and users, while Braintree handles secure payments for a seamless shopping experience.

💻 Tech Stack
Frontend: React.js, Bootstrap
Backend: Express.js, MongoDB
AI: Chatbot for plant care and user queries
Payments: Braintree API
